# OCP Roadmap

## Goal

The first milestone is not "build a product."

The first milestone is "define a simulation constitution that can survive long-term implementation."

## Phase 0: Protocol Foundation

Current focus:

- establish founding vision
- define immutable constitutional layer
- define RFC process
- define repository structure
- reserve protocol vocabulary

Deliverables:

- `README.md`
- `VISION.md`
- `ROADMAP.md`
- RFC index and starter drafts

## Phase 1: Minimal Artificial World

Scope:

- universe persistence
- physics constraints
- energy accounting
- basic life loop
- bounded cognition
- bounded compute budgets
- around 1000 simulated residents
- CLI runtime only

Success criteria:

- no entity can create matter, energy, or knowledge from nothing
- all state transitions have causal traces
- AI residents submit intents instead of mutating state directly
- the world can continue after local civilization collapse

## Phase 2: Emergent Pre-Civilization

Scope:

- agriculture
- reproduction
- trade
- kinship
- tribe formation
- local maps
- first web views

Success criteria:

- surplus and scarcity produce differentiated behavior
- mobility changes cultural contact rates
- groups form and dissolve without hardcoded government templates

## Phase 3: Player Influence Layer

Scope:

- house abstraction
- influence budget
- indirect intervention
- patronage systems

Success criteria:

- players can shape outcomes without direct control
- player power remains protocol-bounded

## Phase 4: Story Discovery

Scope:

- event extraction
- biography generation
- civilization daily reports
- historical summarization

Success criteria:

- stories are discovered from simulation traces
- narrative never overrides causality

## Phase 5: Open Protocol Surface

Scope:

- public API
- SDK
- protocol versioning
- external experimentation surface

Success criteria:

- third parties can inspect or extend the world without breaking constitutional guarantees

## Phase 6: Governance and Platform

Scope:

- DAO processes
- plugin system
- tokenized platform rights

Success criteria:

- governance acts on platform rules without corrupting world integrity

## Recommended Next Build Steps

1. Turn the constitution into machine-readable protocol rules.
2. Define the minimal world state schema.
3. Define the intent-action-resolution loop for AI residents.
4. Define energy and matter accounting invariants.
5. Build a CLI world tick runner before any web UX.
