# RFC-0004 Life Engine

## Status

Draft

## Problem

Life must have a minimal, protocol-defined loop that supports emergence without requiring scripted characters or story logic.

## Defines

- survival loop
- reproduction loop
- adaptation loop
- physiological constraints
- individual-level decision surfaces
