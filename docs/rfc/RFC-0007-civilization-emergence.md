# RFC-0007 Civilization Emergence

## Status

Draft

## Problem

If civilization structures are hardcoded, the project stops being a search system and becomes a disguised historical simulator.

## Defines

- emergence boundaries
- organization formation
- scaling pressures
- collapse and legacy
- activity radius as a major growth variable

## Open Questions

- What is the minimal threshold between group behavior and civilization?
- Which emergent structures should the engine detect versus define?
