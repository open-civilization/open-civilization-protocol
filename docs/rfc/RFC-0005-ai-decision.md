# RFC-0005 AI Decision

## Status

Draft

## Problem

AI should act as a bounded resident that proposes intent, not as an omniscient controller that mutates the world directly.

## Defines

- intent format
- observation limits
- cognition budget
- action proposal
- validation and resolution boundary

## Open Questions

- What should the resident intent schema look like?
- Where should deterministic rules end and LLM support begin?
